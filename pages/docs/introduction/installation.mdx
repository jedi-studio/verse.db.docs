# Installation of the local database

<Steps>
### Install the database

how to Get started with verse.db

```bash copy npm2yarn
npm install verse.db
```

### Set up the database in you code:

- for the `adapter` until now we have `json`, `yaml` and `sql` adapters and we will make more with the next updates.
- `dataPath` the path to the folder of the data 
- `devLogs` the path to the folder of the logs 
- `encryption` the path to the folder of the encryption 
- `backup` the path to the folder of the backup 
 
<Callout type="info" emoji="â„¹ï¸">
  `dataPath` is for this connection or all of them as you want but we recommand that if you are going to make a multiple connections to speret them each connection in a folder or something like `./data/adapter`.
</Callout>
<Callout type="error" emoji="ï¸ðŸš«">
  We are still working on `encryption` & `backup`.
</Callout>


```js copy showLineNumbers {1,3,10-11}
const versedb = require('verse.db')
// or
import versedb from "versedb";

// Connect to the database
const db = versedb.connect({
    adapter: 'json',
    dataPath: './data',
    devLogs: {enable: true, path: "./logs},
    encryption: {enable: false, secret: ""}, // Under Development
    backup: {enable: false, path: "", password: ""}, // Under Development
})
```
